# 📚 MASTER READING APPROACH — Md. Ariful Islam
### The repeatable system for every Academic Reading passage · Target Band 7.0

> **Mindset:** IELTS Reading is a SEARCH-AND-MATCH puzzle, not a comprehension test.
> You don't read to understand — you HUNT for the one sentence that holds the answer,
> and you check whether they paraphrased or trapped it. Memorise NOTHING. Build skill.

---

## ⏱️ MACRO PLAN (60 min · 3 passages · 40 questions)
- ~**20 min per passage**; Passage 3 is hardest — bank time from Passage 1 if you can.
- **Computer-delivered:** NO 10-min answer transfer → use that time to recheck flagged Qs.
  Use the on-screen **highlighter** + **flag/review** tool.
- **Never leave a blank.** No negative marking → always guess all 40.

---

## 🔁 THE 4-STEP LOOP (every passage)
1. **Skim 2–3 min MAX** — title + first sentence of each paragraph + last line. Build a MAP
   ("Para A = history, B = problem, C = solution"). Not understanding — a table of contents.
2. **Read the QUESTIONS, not the passage.** Underline the keyword to scan for (name, date,
   number, CAPITAL word, technical noun — things that DON'T get paraphrased).
3. **Scan → Locate.** Find the keyword's sentence. ONLY here do you read slowly (2–3 sentences).
4. **Verify the trap.** Did they swap a word? add "all/always/never"? "may" vs "will"? Answer, move on.

---

## 🧩 QUESTION-TYPE CHEAT SHEET
| Type | Winning trick |
|---|---|
| True / False / Not Given | TRUE=confirmed · FALSE=contradicted · NG=silent. Use ONLY printed text, never logic/inference. Can't find it → NG. |
| Yes / No / Not Given | Same, but about the WRITER'S opinion/claim. |
| Matching Headings | Pick the paragraph's MAIN idea, not a one-sentence detail. Eliminate. Do easy paras first. |
| Summary / Note completion | Answers come IN ORDER. Grammar-first. Word box → classify part of speech. |
| MCQ | Locate first, THEN read options. Distractors reuse real words but say something slightly wrong. |
| Matching info → paragraph | Do LAST (scattered, time-eating). |
| Sentence completion / Short answer | Obey word limit. Copy EXACT word — spelling counts. |

---

## 🧠 THE 5 BAND-7 KILLERS
1. **Paraphrase blindness** — scanned the exact word; passage used a synonym. (#1 cause.)
2. **True vs Not Given** — you inferred. IELTS punishes inference. Not stated = NG.
3. **Extreme words** — Q: "ALL agree"; passage: "MANY agree" → FALSE. Watch all/always/never/only/first.
4. **Distractor near the answer** — right paragraph also holds a tempting wrong detail.
5. **Time bleed** — 4 min on one hard Q. Flag → guess → move on → return.

---

## 📓 NOTE / ERROR-LOG SYSTEM (this is what improves you)
Don't note passage topics. Note only THREE things:
- **A. Paraphrase pairs** — every miss: `dominated → influence`, `faithful → devoted`. (Highest value.)
- **B. Error TYPE code** — tag every wrong answer:
  `KW` wrong keyword · `PP` missed paraphrase · `NG` T/F/NG inference error ·
  `EX` extreme-word trap · `TM` time/rush · `VC` vocab gap · `LIM` word-limit/spelling.
  → After a few passages ONE code dominates = your real weakness. Fix that one.
- **C. Blocking vocab only** → app vocab deck with Bangla + synonym.

---

## ✅ DO / ❌ DON'T
**DO:** skim for a map · scan for keywords · read deeply ONLY at the answer · trust the text literally ·
fill every blank · flag & return · log error TYPES.
**DON'T:** read every word · use outside knowledge · pick "looks right" without proof · argue NG into TRUE ·
burn 4 min on one Q · re-read the whole passage per question · take topic notes.
