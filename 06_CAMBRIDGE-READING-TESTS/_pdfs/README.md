# Put your own Cambridge IELTS PDFs here

Name each file by book number so the test pages can find it:

    Cam-11.pdf  Cam-12.pdf  Cam-13.pdf  ...  Cam-20.pdf

Once a file is here, open that book's test page (e.g. Cam-15/Cam-15-Test-1.html)
and the passage + questions show up INSIDE the page (your own PDF, displayed locally).
Use the page box on the test page to jump to the right Reading passage page.

These are YOUR legally-owned book files — they stay on your computer, nothing is uploaded.
