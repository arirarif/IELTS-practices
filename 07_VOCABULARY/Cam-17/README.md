## কীভাবে ব্যবহার করবে
1. রোজ `Cam-17-Vocabulary.html` খুলে **search** করে দ্রুত revise করো (word/বাংলা/উচ্চারণ এক লাইনে)।

3. শব্দ মুখস্থের পর `02–05` ফাইলগুলো ধরে idiom/phrasal/collocation/sentence frame শেখো।
4. এরপর ওই test-এর passage solve করতে গেলে শব্দগুলো চিনে যাবে — Cambridge standard বলে অন্য বই/exam-এও ফিরে আসবে।
