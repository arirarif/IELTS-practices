## 🔁 The Daily Ritual (do this every single day)

```
1. Open 08-WEEK-BY-WEEK.md  → find today's task
2. Open the relevant playbook (02–05) → re-read the "Golden Rules" box
3. Do the work (timed, like the real exam — always timed)
4. Mark every mistake → log patterns in 07-PROGRESS-TRACKER.md
5. Tick today off in 08-WEEK-BY-WEEK.md
```

> ⚠️ **The #1 rule: ALWAYS practice timed.** Band 7 is lost on the clock, not on knowledge.
