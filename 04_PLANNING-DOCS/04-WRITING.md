# ✍️ 04 — WRITING PLAYBOOK (Academic)

> **Target: Band 6.5** (a safe, realistic anchor — anything above is bonus). Writing is the slowest to move, so we use **reliable structures** so you never freeze, and we **mark every essay examiner-strict**.

You write **two tasks in 60 minutes**:
- **Task 1** (20 min, ≥150 words) — describe a graph/chart/table/process. *Worth 1/3 of the writing score.*
- **Task 2** (40 min, ≥250 words) — an opinion/discussion essay. *Worth 2/3 — prioritize it.*

---

## 📊 The 4 Marking Criteria (each = 25%)

| Criterion | What the examiner checks |
|---|---|
| **Task Achievement/Response** | Did you answer the *exact* question fully, with a clear position + developed ideas? |
| **Coherence & Cohesion** | Logical paragraphs, clear progression, linking words used *naturally* (not stuffed). |
| **Lexical Resource** | Range + accuracy of vocabulary; good paraphrasing; not repeating the same words. |
| **Grammatical Range & Accuracy** | Mix of sentence types; correct grammar; punctuation. |

> 🔴 Two cheap ways people lose a band: **under word count** (auto-penalty) and **not answering the actual question asked** (huge Task Response hit). Always count, always re-read the prompt.

# 🟩 TASK 1 — Describing Visuals (20 min, ≥150 words)

### The bulletproof 4-paragraph structure
```
¶1  INTRODUCTION  → Paraphrase what the graph shows (don't copy the prompt).
¶2  OVERVIEW      → The 2–3 biggest features, NO numbers. ← THIS is what makes it Band 7.
¶3  BODY 1        → Detailed data for one group of features (with figures).
¶4  BODY 2        → Detailed data for the rest (with figures + comparisons).
```

### 🥇 The Overview is the Band 6 → 7 line
- Start it with **"Overall,"** — examiners reward the signpost.
- State the **headline trends** (highest, lowest, biggest change, overall direction) — **no specific numbers** here.
- A response with **no clear overview caps at Band 5.** This paragraph is non-negotiable.

### Golden rules
1. ❌ **Never give opinions or reasons** ("this may be because…"). Task 1 = *factual description only.*
2. 🔢 **Select key features — don't report every number.** Examiners reward smart selection, not a data dump.
3. 📐 Aim **170–190 words** (enough detail, still controlled).
4. 🔤 Vocabulary by chart type:
   - **Trends over time:** rose, climbed, surged, dipped, plummeted, fluctuated, plateaued, peaked.
   - **Comparisons (single point):** higher/lower than, twice as much, the majority, a minority, accounts for.
   - **Process diagram:** first, next, subsequently, following this, finally + **passive voice** ("the water is filtered").
5. 📏 Vary how you express data: *rose to 50% / a rise of 20 points / doubled / a tenfold increase.*

# 🟦 TASK 2 — The Essay (40 min, ≥250 words)

### The 4-paragraph structure (works for every prompt type)
```
¶1  INTRODUCTION → Paraphrase the topic + state your CLEAR position/answer.
¶2  BODY 1       → Main idea 1  (PEEL)
¶3  BODY 2       → Main idea 2  (PEEL)
¶4  CONCLUSION   → Restate position + summarize. (No new ideas.)
```

### 🧱 Each body paragraph = PEEL
| Letter | Do |
|---|---|
| **P** — Point | Topic sentence: one clear main idea. |
| **E** — Explain | Develop it — *why*, *how*. |
| **E** — Example | A specific, concrete example (real or realistic). |
| **L** — Link | Tie back to the question / lead to next paragraph. |

### Know your prompt type (answer the *right* question)
| Prompt type | What it demands |
|---|---|
| **Opinion** (do you agree/disagree) | Pick a side, state it clearly in ¶1, defend it throughout. |
| **Discussion** (discuss both views + your opinion) | One view per body paragraph + **your own opinion stated clearly.** |
| **Advantages/Disadvantages** | One each, or weigh them — match what the prompt asks. |
| **Problem/Solution (or Cause/Solution)** | One paragraph problems/causes, one solutions. |
| **Two-part question** | Answer **both** parts — missing one wrecks Task Response. |

### Golden rules
1. 🎯 **Make your position crystal clear in the introduction**, right after the topic.
2. 🔗 **Don't stuff linkers.** One linker per *sentence* is too many — it reads mechanical and *lowers* your score. Use them where logic needs them.
3. 🗣️ **Drop memorized template phrases** ("In this globalized era of modern technology…"). Examiners spot them instantly → Task Response penalty. Write to the *specific* prompt in your own words.
4. ✍️ **Topic sentence first** in every body paragraph — tells the examiner exactly what's coming.
5. 📝 **Plan for 5 minutes.** Decide your position + 2 main ideas + examples *before* writing. Saves you from rambling.
6. 🔢 Aim **260–290 words.** Under 250 = penalty; way over = error-prone and time-wasting.
7. ⏰ **Leave 3–4 min to proofread** — catch articles (a/the), subject-verb agreement, plurals, tense.

## 🏋️ Weekly Writing Routine
- **2–3 essays/week** (mix Task 1 + Task 2), every one **timed.**
- **Every essay marked examiner-strict** (by me) → exact band per criterion + fixes.
- **Rewrite** each essay using the feedback. *One marked-and-rewritten essay beats five unmarked ones.*
- Build a personal bank of: linking phrases, paraphrase patterns, Task 1 trend verbs, flexible examples.
