### PHASE 3 — Find your weaknesses via FAST practice (Tire 2, Goal 2 → Tire 3)

### PHASE 5 — Re-test, re-diagnose, repeat (the loop)
- [ ] Do another real test → check improvement → find **new** weaknesses → fix. Loop.

### PHASE 6 — Only now, book the test (Readiness rule)
- [ ] Book **only** when you score **half a band above target, 3 times in a row.** (For Band 7 target → **7.5+ three times running.**)
- [ ] Remember: #1 failure cause is **booking too early.**

## ✅ THE TEST-DAY EXECUTION CHECKLIST
- [ ] **Per section:** check which question types appear → do **Matching Headings FIRST**, **Matching Information LAST**.
- [ ] **Spend less time on Section 1** (fast + accurate) → bank time for Section 3.
- [ ] **Every question:** read the **instructions** (word/number limit!) → **read the question/statement fully** (not just keywords) → **predict** (where useful) → think **synonyms** → **"where before what"** (locate first) → **close-read** → decide.
- [ ] **Completion/table/diagram:** transfer words **exactly**, obey the **word limit**, **check spelling**.
- [ ] **TFNG/YNNG:** check fact (TFNG) vs opinion (YNNG); not there → **Not Given** (have the confidence!); watch **qualifying words**.
- [ ] **MCQ:** understand option **differences**; **eliminate** wrong ones; keyword = location, not answer.
- [ ] **Stuck?** Mark a guess (e.g. "NG"), **move on**, come back. Don't burn 5–10 min on one item.
- [ ] **Unknown word?** Guess from context; ask "do I even need it?" → usually move on. Don't panic.

## 📌 THE COMPLETE STRATEGY INDEX (quick-reference of every rule)
**Universal rules:** Where before what · Be aware of synonyms · Read questions before text · Predict before reading · Read instructions every time · Mind word count & spelling · Confidence to move on · It's a vocabulary test too.
**Skills:** Skim (general meaning) · Scan (locate) · Close read (decide) · Active > passive reading · Guess meaning from context.
**Ordering:** Matching Headings first · Matching Information last · Section 1 fast.
**Per-type quick map:**
- Sentence/Summary completion → read sentence first, predict (grammar for summary), synonyms, locate, close-read, spelling.
- MCQ → understand option differences, skim, locate, eliminate, decide.
- Short answer → understand question, underline nouns, synonyms, locate, re-read question.
- Diagram → don't panic, study it, predict, locate, spelling.
- TFNG → fact; whole sentence; qualifiers; locate; close-read; T/F/NG; confidence for NG.
- YNNG → writer's opinion (else same as TFNG); beware other people's opinions.
- Matching sentence endings → predict endings, match obvious, eliminate, base on TEXT not logic.
- Matching names → scan & underline names, do single-mention names first, delete matched.
- Matching information → do LAST, skim text, locate, move on if stuck.
- Table/flowchart completion → read instructions, locate, read carefully, copy exactly, spelling.
- Matching headings → do FIRST, ignore headings, write your own heading, then match, close-read rest.
