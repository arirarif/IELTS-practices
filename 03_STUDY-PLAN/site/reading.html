03 · Point bank #1 (fastest ROI)

Reading Playbook

Pure technique + speed + timing — the most trainable skill in 8 weeks. 60 min · 3 passages · 40 questions. Band 7 ≈ 30/40, Band 8 ≈ 35/40. Tools: Simon Master Reading + Cambridge 16–20 + app drills.

The golden rule You do NOT read the passage fully first — you don't have time.
  1. Skim ~2–3 min: title, first/last sentence of each paragraph → gist + structure.
  2. Read the questions, then scan for keyword synonyms.
  3. Answers usually come in order (except matching/heading types).
  4. ~20 min per passage. Never leave a blank — guess.

Question-type tactics

TypeMethod
TRUE / FALSE / NOT GIVENTRUE = passage confirms · FALSE = passage contradicts · NOT GIVEN = passage is silent. If you must reason/assume to make it true → it's NOT GIVEN. No outside knowledge.
YES / NO / NOT GIVENSame logic, about the writer's opinion/claim.
Matching HeadingsFind each paragraph's main idea (not a detail). Do these last. Cross off used headings.
Matching InformationScan-only; can be anywhere; time-eater → do later.
MCQRead all options; eliminate wrong; beware "true but doesn't answer the question."
Gap / sentence / summary completionObey the word limit; copy spelling exactly; answer must fit grammatically.
Spelling counts (computer-delivered) A misspelt right answer is wrong. Copy from the text precisely; match singular/plural.

Speed training

  • Time every passage. Slow? The fix is usually reading too much — drill skim/scan.
  • Build a synonym sense: the answer is rarely the same word as the question (question "children" → text "youngsters"). Log synonym pairs you miss.

How to practise (the loop)

  1. App reading drill daily (fast per-type feedback).
  2. Cambridge full reading paper weekly, strictly 60 min, on screen.
  3. For each wrong answer, identify the type and the why (misread? synonym missed? time? NG vs F?). Log it.
  4. Drill your 2 weakest types until accuracy > 80%.

✅ Do

  • Skim then scan
  • Trust "answers in order"
  • Obey word limits
  • Guess every blank
  • Watch the clock per passage

❌ Don't

  • Read every word
  • Use outside knowledge on TFNG
  • Spend 30 min on passage 1
  • Leave blanks
  • Ignore spelling
